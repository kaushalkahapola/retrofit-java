SUPPORT & TROUBLESHOOTING
================================================================================

Common Issues:
  1. "MCP server connection refused"
     → Check if server is running: McpClient.isHealthy()
     → Verify MCP_SERVER_URL environment variable
  
  2. "Signature not found"
     → Try wildcard: "method(...)" instead of exact signature
     → Verify method exists first: get_method_boundaries()
  
  3. "Cache limit exceeded"
     → Increase CACHE_SIZE_LIMIT (Java) or cache_ttl (Python)
     → Clear cache if needed: ReplaceMethodBodyTool.clearCache()
  
  4. "Slow performance"
     → Check cache stats: getCacheStats()
     → Increase connection pool limits
     → Profile network latency to MCP server

Contact Points:
  - Java MCP Server: http://localhost:8080/mcp/sync/call
  - Health endpoint: http://localhost:8080/mcp/sync/call (HEAD request)
  - Python client logs: Check McpClientError exceptions
